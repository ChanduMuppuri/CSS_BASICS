CSS Positioning Techniques

Static Positioning

This element is positioned statically. It is the default positioning for all elements.

Relative Positioning

This element is positioned relatively. It is offset by 20px from its normal position.

Absolute Positioning

This element is positioned absolutely. It is placed 50px from the top and left of its containing block.

Fixed Positioning

This element is positioned fixed. It is placed 0px from the top and right of the viewport and stays in place when the page is scrolled.

Sticky Positioning

This element is positioned sticky. It will stick to the top of the viewport when you scroll past it.
